# リファレンス

Yuライブラリが提供するサブライブラリのリファレンスの一覧です．

Yuライブラリでは，リファレンスを仕様とした，仕様先行型の開発手法を採用しています．
そのため，リファレンスには未実装の機能も記述されています．
詳細は[設計文書 > 開発手法](/design/development-style.md)を参照してください．
